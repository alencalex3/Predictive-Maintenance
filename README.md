Predictive Maintenance using Machine Learning, SQL & Power BI
 Project Overview

This project predicts machine failures using Machine Learning and presents insights through an interactive Power BI dashboard. It combines Python for data analysis and model building, SQL for data querying, and Power BI for visualization.


 Objectives
Analyze machine sensor data.
Predict machine failures.
Identify failure patterns.
Build an interactive dashboard for business insights.
