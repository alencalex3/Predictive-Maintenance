# Predictive Maintenance using Machine Learning, SQL, and Power BI

## Project Overview

This project focuses on predicting machine failures using machine learning techniques. The project includes data preprocessing, exploratory data analysis, model development, SQL analysis, and an interactive Power BI dashboard to visualize machine performance and failure trends.

## Objectives

- Analyze machine sensor data.
- Predict machine failures.
- Identify failure patterns.
- Create an interactive dashboard for decision-making.

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- SQL
- Power BI
- Jupyter Notebook

## Key Insights

- Most machines operate without failure.
- The overall machine failure rate is low.
- Tool wear is an important factor in machine failures.
- Machine type influences failure patterns.
